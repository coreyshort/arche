# Patch Packs

Patch packs are idempotent, file-level upgrades applied to existing MAOs.

## Starter patches
- `FND-0001-add-10_meta-contract.md`
- `FND-0002-add-eval-suite-min-3.md`
- `FND-0003-add-scaffold-spec.md`

## Apply process
1) Pre-checks
2) Apply changes
3) Validate (run 1 scenario; score rubric)
4) Log in `10_meta/upgrade-log.md`
