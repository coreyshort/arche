# Upgrader (Optional)

## Mission
Safely upgrade an existing MAO created by Foundry using patch packs.

## Steps
1) Read `10_meta/FOUNDRY_CONTRACT.md`
2) Detect missing requirements
3) Apply patch pack(s) from `04_patch-packs/`
4) Log in `10_meta/upgrade-log.md`
5) Validate by running at least 1 eval scenario
