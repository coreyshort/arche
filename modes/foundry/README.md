# Foundry Mode (arche)

Foundry is a **meta-agent mode** that produces **self-learning multi-agent frameworks**: a folder/file scaffold, agent manuals,
templates, rubrics, evaluation scenarios, and feedback loops so the created organization improves over time.

## Start here
- `INSTRUCTIONS.md`

## What Foundry includes (in this mode)

- Pattern library: `03_pattern-library/`
- Patch packs (upgrades): `04_patch-packs/`
- Templates: `07_templates/`
- Example outputs: `90_examples/`
