# Quick Actions

**To apply these settings:**

1. Go to https://github.com/coreyshort/arche
2. Click "Settings" (gear icon near top right)
3. Scroll to "About" section in right sidebar (or repository homepage)
4. Click "Edit repository details" (gear icon)
5. Paste description into "Description" field
6. Paste topics (comma-separated) into "Topics" field
7. Add website URL
8. Check "Releases" and "Packages" if desired
9. Save changes

**Result:** Arche will immediately be more discoverable in GitHub search and will show these tags on the repository homepage.
